# PIng-pong.1
Игра Пинг-понг созданная вручную на пайтон.Имеет управление (двухпользовательский режим): Игрок 1 (слева): Клавиши w (вверх) и s (вниз). Игрок 2 (справа): Клавиши Up (стрелка вверх) и Down (стрелка вниз). Логика: Включает движение мяча, отскоки от ракеток и верхней/нижней границ, Если мяч прошел за ракетку то +балл 1 из сторон
